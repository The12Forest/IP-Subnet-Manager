# Documentation

This folder contains the complete documentation for the Subnet Manager application.

- **[SETUP.md](./SETUP.md)**: Detailed instructions on how to set up and run the application using Docker, including environment variables and HTTPS configuration.
- **[API.md](./API.md)**: A complete reference for the backend REST API.
- **[MCP.md](./MCP.md)**: A guide for connecting and using the built-in MCP (Managed Component Protocol) server.
- **[DEVELOPMENT.md](./DEVELOPMENT.md)**: Instructions for setting up a local development environment without Docker.
- **[ARCHITECTURE.md](./ARCHITECTURE.md)**: An overview of the application's technical architecture.
