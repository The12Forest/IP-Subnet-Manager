Claude.ai Integration — Setup Steps

  1. Expose port 3001 publicly (tunnel, reverse proxy, or port forward)
  2. Open claude.ai → Settings → Integrations → Add integration
  3. Paste the MCP URL and enter the OAuth Client ID & Secret below
  4. Click Add — Claude will connect and list available tools
Replace with your public domain or tunnel URL
